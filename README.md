## 레슨 추가하는 법 (찬돌쌤용)

`js/curriculum.js` 파일 하나만 고치면 됩니다. 파일 맨 위에 패턴 쓰는 법이 주석으로 적혀 있어요.

- 한 마디 = 16칸(16분음표), `x` 치기 / `X` 세게 / `g` 고스트 / `-` 쉼 / `|` 마디 구분
- 레인: `HH` 하이햇, `SD` 스네어, `BD` 킥, `T1` 하이탐, `T2` 플로어탐, `CR` 크래시, `RD` 라이드, `OH` 오픈하이햇, `ST` 손 순서(R/L)

## 구조

| 파일 | 역할 |
|---|---|
| `js/curriculum.js` | 커리큘럼(코스·레슨·패턴) — 콘텐츠는 여기만 |
| `js/pattern.js` | 패턴 문자열 → 데이터 변환 |
| `js/notation.js` | 드럼 오선보 SVG 렌더링 |
| `js/audio.js` | 웹오디오 드럼 사운드 합성 + 메트로놈 + 시퀀서 |
| `js/app.js` | 화면 연결(재생·BPM·완료 체크) |

빌드 과정 없음 — 정적 파일 그대로 GitHub Pages에서 서빙.
